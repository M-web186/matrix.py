 ## matrix.py

A simple Python library for matrix operations.

## Install

```bash
pip install matrix.py
Usage
Python
from matrix import Matrix

A = Matrix([[1, 2], [3, 4]])
B = Matrix([[5, 6], [7, 8]])

C = A + B      # Add
D = A * B      # Multiply
E = A.transpose()
det = A.determinant()
